Fedora Core 5 セットアップ

Fedora Core 5 のインストールを行う

OSのインストール関しては、下記の注意点以外は特に行う事はないので自由にインストールを行ってください。筆者の環境では録画ファイルを溜め込むことから、録画フォルダをシステムとは、別HDD別パーティションにするという方法を取っています。

インストール時の注意

  • 開発ツールは最低限入れておく

システム更新

最新の状態にするため、アップデートを行う

# yum update

foltiaユーザ作成

foltiaを動かす為のユーザを作成し、パスワードの設定を行う

# useradd foltia
# passwd foltia
